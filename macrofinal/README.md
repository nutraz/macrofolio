# Macrofolio (macrofinal)

Single, judge-ready project folder for the RevenueCat Shipyard Creator contest.

## What’s inside

- `web/` — Vite + React web app (Vercel-ready)
- `mobile/` — React Native app
- `backend/` — Supabase schema
- `chain/` — Hardhat + `PortfolioAnchor.sol`
- `docs/` — submission docs + setup guides
- `scripts/` — helper scripts (Android/iOS)

## Quick start (web)

```bash
cd web
npm install
cp .env.example .env
npm run dev
```
